AI-Powered Customer Churn Intelligence Platform
An end-to-end Machine Learning and Business Intelligence solution designed to predict customer churn and support data-driven decision-making in the banking sector. This project combines advanced customer analytics, interactive data visualizations, predictive modeling, and API integration to identify at-risk customers, analyze behavioral patterns, and enable proactive retention strategies.
Built using Python, Scikit-Learn, Streamlit, Plotly, and FastAPI, the platform provides real-time churn predictions, executive-level business insights, customer segmentation analysis, and an intuitive dashboard experience. The solution helps organizations reduce customer attrition, improve retention rates, and optimize business performance through actionable intelligence.
